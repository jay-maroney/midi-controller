# Custom MIDI Keyboard Controller

Designing a custom MIDI keyboard built from first principles.

## Goals

- Design velocity-sensitive keybed
- Implement microcontroller-based key scanning
- Develop USB-MIDI communication
- Design custom PCB

## System Overview

The controller includes:

- Key matrix scanning circuitry
- Velocity detection sensors
- Microcontroller firmware
- USB MIDI interface
- Custom PCB for signal routing

Embedded Systems  
Circuit Design  
Digital Signal Processing
